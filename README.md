# NerdStoreEnterprise
NerdStoreEnterprise is an e-commerce for the nerd public

## Technologies / Components implemented

- .NET 6
    - ASP.NET MVC Core
    - ASP.NET WebApi
    - ASP.NET Minimal API
    - ASP.NET Identity Core
    - JWT    
    - Background Services
    - Entity Framework Core 6

- Components / Services
    - RabbitMQ
    - EasyNetQ
    - Refit 
    - Polly
    - Dapper
    - FluentValidator
    - MediatR
    - Swagger UI with JWT support
